# PE-IV-front
Front-end do Projeto de Extensão IV


### Instruções para rodar o projeto:

- Em um terminal, rodar 'npm run dev' na subpasta PE-IV-FRONTEND/backend-mock .

- Em outro, rodar 'npm start' na pasta do projeto /PE-IV-FRONTEND .

- No login utilizar: usuário: 'Diretor' e senha 'admin3'.
